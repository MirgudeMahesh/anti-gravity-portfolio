# Anti-Gravity Portfolio

A premium, high-performance portfolio website built with React, Vite, and Framer Motion.

## Features
- **Minimalist Geometric Design**: Deep slate background with technical grid and cyan accents.
- **AI-Integrated Projects**: Simulated AI analysis for project cards.
- **Glassmorphism UI**: Modern frosted glass effects.
- **Responsive**: Fully mobile-optimized layout.
